Mangrr (Project Mango) Versions
Version 0.1.7
Released in : February 21, 2019
New Features
  • Added Group types (Open, Public, Private) in the Group Edit page.
  • Added "Follow" button in Public Group pages.
  • Added "Join" button in Open Group pages.
  • Added "Invite To Group" button in User pages (available if the user is owner of one or more groups).
Changes
  • Removed the "is Private" option from the Group Settings Page
Bug Fixes
  • Fixed an issue where pressing the "Request To Join" button in Groups would take longer than expected to react.
  • Fixed an issue where friend requests duplicate in the recepient's notification menu.
  • Fixed an issue where posting in a private group would create duplicate posts.
Version 0.1.6
Released in : February 19, 2019
Changes
  • Registered domain name mangrr.com
  • Website branding changed from "Mango" to "Mangrr"
  • The option to select a Display Name while creating an account in the Register page.
Bug Fixes
  • Fixed an issue where new users cannot access the home page at times.
  • Fixed an issue where video buffering seemed faster than it actually is.
  • Fixed an issue where uploading links with "?" simbols resulted in the posted link to duplicate.
Version 0.1.5
Released in : February 16, 2019
New Features
  • Added Groups in Profile Navigation.
  • Added Group page.
  • Added Manage Groups page.
  • Added New Group page.
  • Added Edit Group page.
  • Added Versions page.
Changes
  • Profile Groups are now visible in the Home and Profile page.
  • Posts can now recognize hyperlinks and make them accessible.
  • Group posts display in the post feed of the home page.
Bug Fixes
  • Improved performance and stability.
  • Fixed an issue where upon creating posts a Form Resubmission alert would appear.
  • Fixed an issue where posts are duplicted upon posting.
  • Fixed an issue where leaving the content section of a new post blank would prevent the user from creating a new post.
