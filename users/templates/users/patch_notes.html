Pinbitt Versions
Version 0.1.9
Released in : March 4, 2019
New Features
  • Added Friends Messaging (Available next to the Notifications bell icon.).
  • Added Friends Messaging for the mobile web version of Pinbitt.
  • Added 'MP3' audio support for Posts.
Changes
  • Changed "Add Photo or Video" to "Add Media" button when uploading a new post.
  • Users can now upload 'PNG' formatted images in posts.
  • 404 and 500 errors no longer show the application's backend log. Instead a new Error Page will load.
Bug Fixes
  • Fixed an issue where certain actions in the mobile version would result in images scaling improperly.
  • Fixed an issue where new notifications would take mora than 10 seconds to load.
  • Fixed an issue where clicking on a video multiple times results in it buffering again.
  • Fixed an issue where topics from private groups appear in the "Topics of the Day" panel.
  • Fixed an issue where typing strange character combinations in the search bar would cause a 400 error.
  • Slight improvements to refresh speeds.
Version 0.1.8
Released in : February 26, 2019
New Features
  • Added a mobile friendly view mode when using Pinbitt in a mobile browser.
  • Added anonymity settings for Groups.
Changes
  • A loading animation will play when uploading images and video files.
  • users can now upload animated GIFs as a profile picture.
  • JPEG image format now supported.
Bug Fixes
  • Fixed an issue where the New Post pop up stretches vertically when using low resolution displays.
  • Fixed an issue where inviting someone to the same group multiple times in a short amount of time was possible.
  • Fixed an issue where clicking on the Post button multiple times when creating a new Post ends up in uploading duplicate posts.
Version 0.1.7
Released in : February 21, 2019
New Features
  • Added Group types (Open, Public, Private) in the Group Edit page.
  • Added "Follow" button in Public Group pages.
  • Added "Join" button in Open Group pages.
  • Added "Invite To Group" button in User pages (available if the user is owner of one or more groups).
Changes
  • Removed the "Private Group" option from the Group Settings Page
Bug Fixes
  • Fixed an issue where pressing the "Request To Join" button in Groups would take longer than expected to react.
  • Fixed an issue where friend requests duplicate in the recepient's notification menu.
  • Fixed an issue where posting in a private group would create duplicate posts.
Version 0.1.6
Released in : February 19, 2019
Changes
  • Registered domain name pinbitt.com
  • Website branding changed from "Mango" to "Pinbitt"
  • The option to select a Display Name while creating an account in the Register page.
Bug Fixes
  • Fixed an issue where new users cannot access the home page at times.
  • Fixed an issue where video buffering seemed faster than it actually is.
  • Fixed an issue where uploading links with "?" simbols resulted in the posted link to duplicate.
Version 0.1.5
Released in : February 16, 2019
New Features
  • Added Groups in Profile Navigation.
  • Added Group page.
  • Added Manage Groups page.
  • Added New Group page.
  • Added Edit Group page.
  • Added Versions page.
Changes
  • Profile Groups are now visible in the Home and Profile page.
  • Posts can now recognize hyperlinks and make them accessible.
  • Group posts display in the post feed of the home page.
Bug Fixes
  • Improved performance and stability.
  • Fixed an issue where upon creating posts a Form Resubmission alert would appear.
  • Fixed an issue where posts are duplicted upon posting.
  • Fixed an issue where leaving the content section of a new post blank would prevent the user from creating a new post.
